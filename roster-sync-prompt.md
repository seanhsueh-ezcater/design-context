# Roster Sync — Design Prompt

Let the admin choose to set up a standard roster sync process where all US-based employees are automatically ported into the ezCater roster, or a custom roster sync process.

For the custom roster sync process, let the admin choose to include employees using a combination of 5 distinct attributes: location, team, region, department, and email domain.

Let the ezCater admin select which employee attributes to use, and also let them indicate the right input for each attribute in an easy method (for example, writing that the desired employee region is North America.)

Give the admin confirmation and a preview of their choices before they confirm the roster sync, indicating how many SAP Concur employees will be synced into the ezCater enterprise account, and how many will be removed.
